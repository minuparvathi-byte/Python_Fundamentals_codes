# Python Case Studies for Data Analytics Learning

## 📌 Project Overview

This repository contains a collection of beginner-friendly Python case studies designed to strengthen fundamental programming skills and prepare for a career in Data Analytics. The projects focus on real-world scenarios involving user input, data manipulation, string operations, lists, dictionaries, and sets.

These hands-on exercises helped me build a strong foundation in Python while improving my problem-solving and logical thinking abilities.

---

## 📚 Case Studies Included

### 1. Student Registration System

**Concepts Covered:**

* Variables
* User Input (`input()`)
* Type Casting (`int()`, `float()`)
* Arithmetic Operations
* f-Strings
* Data Types (`type()`)

**Key Features:**

* Collects student and course details
* Calculates GST (18%) and total course fee
* Displays student information and data types

---

### 2. Library Fine Calculator

**Concepts Covered:**

* Variables
* User Input
* Type Casting
* Mathematical Calculations
* Formatted Output

**Key Features:**

* Records student and book details
* Calculates total fine based on overdue days
* Displays details and data types

---

### 3. Email Username Generator

**Concepts Covered:**

* String Manipulation
* `lower()`
* `replace()`
* String Formatting

**Key Features:**

* Converts user name into email format
* Generates professional email IDs automatically

---

### 4. Student ID Card Generator

**Concepts Covered:**

* String Methods
* String Slicing
* Concatenation
* User Input

**Key Features:**

* Generates unique student IDs
* Extracts portions of names and departments
* Displays formatted student information

---

### 5. Shopping Cart System

**Concepts Covered:**

* Python Lists
* List Methods
* Indexing
* Slicing

**Key Features:**

* Add, insert, remove, and update cart items
* Find item positions and counts
* Calculate total number of items

---

### 6. Student Data Management System

**Concepts Covered:**

* Dictionaries
* Lists within Dictionaries
* Data Storage and Retrieval

**Key Features:**

* Stores multiple student records
* Adds new student information dynamically
* Retrieves specific student details

### 7. Student Course Registration System

**Concepts Covered:**

* Python Sets
* Set Operations
* Duplicate Handling

**Key Features:**

* Manages unique course registrations
* Demonstrates add, update, remove, and discard operations
